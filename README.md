# Skill Registry Builder


Agent skills are multiplying fast. Building a registry to organize, share, and distribute them is still treated like an engineering problem. It shouldn't be. This skill walks anyone — regardless of technical background or job title — through the decisions that actually matter: how people find skills, how they evaluate them, how they install them, and how the registry stays useful over time.

---

Read more about the philosophy behind this — [The Agent Skills Democratization Problem](https://noa-on-ai.vercel.app/blog/agent-skills-democratization-problem).

## Skills

### `skill-registry-builder`

Guides you through designing and building a skill or agent registry from scratch. Covers four pillars: Discovery, Evaluation, Installation & Use, and Maintenance. Collects your skills through conversation and produces a finished, self-contained `registry.html` website — open it in any browser, share it as-is.

Just say _"help me design a skill registry"_ and it triggers automatically.
